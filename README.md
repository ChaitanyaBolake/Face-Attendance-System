# 🎯 Face Recognition Based Attendance System

An automated attendance management system built using Python and Computer Vision.
The system detects and recognizes faces in real-time using a webcam and automatically records attendance with date and time.

# 📌 Project Objective

This project eliminates manual attendance systems and prevents proxy attendance using facial recognition technology.
It improves:

✅ Accuracy <br>
✅ Efficiency <br>
✅ Security <br>
✅ Automation <br>

# 🚀 Features

Face image capture and dataset creation <br>
Face training and encoding <br>
Real-time face recognition via webcam <br>
Automatic attendance marking <br>
Date & time recording <br>
CSV-based attendance storage <br>
Admin password protection <br>
User-friendly GUI built with Tkinter <br>

#⚙️ System Workflow

Register a new user by capturing face images.
Train the system to encode facial features.
Start real-time face recognition using webcam.
Match detected faces with stored encodings.
Mark attendance automatically in a CSV file.
Save attendance with Name, Date, and Time.


# 💻 Installation & Setup

pip install numpy
pip install pandas
pip install pillow

# ▶️ How to Run
python main.py
Make sure your webcam is connected before running the system.
